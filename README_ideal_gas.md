# N-Particle Ideal Gas Simulation

## Overview
This simulation models the behavior of an ideal gas consisting of N-particles, allowing users to visualize how the particles interact under various conditions.

## Controls
- **Adjust particle count**: Modify the number of particles.
- **Temperature control**: Set the temperature of the gas.

## Features
- 2D/3D visualization of gas particles.
- Interaction between particles with collision handling.
- Energy calculations and graphs.

## Installation
To install, clone the repository and run the script using Python.

```bash
git clone https://github.com/SebsMwendaweli/Astro-simmulation.git
cd Astro-simmulation
python ideal_gas.py
```
